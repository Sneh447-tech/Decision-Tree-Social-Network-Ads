# Social Network Ads Prediction using Decision Tree (CART)

## Project Overview
This project predicts whether a user will purchase a product based on Age and Estimated Salary using a Decision Tree Classifier with the CART algorithm.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset
Social Network Ads dataset containing:
- Age
- Estimated Salary
- Purchased (Target Variable)

## Workflow
1. Import libraries
2. Load dataset
3. Data preprocessing
4. Feature scaling
5. Train-test split
6. Train Decision Tree model
7. Prediction
8. Model evaluation using confusion matrix and accuracy
9. Visualization of decision tree

## Model
Decision Tree Classifier using CART algorithm.

## Output
Decision tree visualization and prediction results.

## Example Prediction
Input:
Age = 19  
Salary = 19000  

Output:
Not Purchased
